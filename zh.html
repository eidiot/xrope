xrope by eidiot

English | 中文
用于原生AS3显示对象的简单layout库。

特点

  • 用于原生AS3显示对象，没有Flex或任何其它依赖。
  • AtomLayout 使用 getBounds() 排列显示对象，不用担心注册点问题。
  • 组合 layout 元素/组 非常容易，不需要额外的显示容器。 (example)

如何使用

  • 创建一个实现 ILayoutGroup 接口的 layout group 并根据需要设置属性。
  • 添加显示对象或其它 ILayoutElement。
  • 调用 layout() 方法

Layout groups (AsDoc)

  • HLineLayout - 水平单行 layout. (example)
  • VLineLayout - 垂直单行 layout. (example)
  • HMultiLineLayout - 水平多行 layout. (example)
  • VMultiLineLayout 垂直多行 layout. (example)
  • HBoxLayout - 水平 box layout. (example)
  • VBoxLayout - 垂直 box layout.(example)
  • AlignLayout - Align 位置 group layout. (example)
*注意 MultiLineLayout 和 BoxLayout 的区别: MultiLineLayout 里所有行的高度都和 lineHeight 属性一样，而 BoxLayout 各行的高度为行中最高元素的高度
