h2. Xrope: Simple layout library for native as3 display objects.

xrope is a lightweight layout library for native as3 display objects. No need for any framework or dependence. It only has simple layout for games and mini-sites. no feature common used in applications like auto-resize is supported.

"Examples":http://eidiot.github.com/xrope/

h3. How to use it.

* Create a layout group which implements _ILayoutGroup_ and set layout properties.
* Add some display objects or other _ILayoutElement_ to the group.
* Call the _layout()_ method of the group.

h3. Done
