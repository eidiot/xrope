h2. Xrope: Simple layout library for native as3 display objects.

"Examples":http://eidiot.github.com/xrope/

h3. Features

* For native as3 display objects. No Flex or any other dependency.
* AtomLayout align DisplayObject using getBounds(). No worries about register point in FLA.
* Easily composite layout elements/groups. No more DisplayObjectContainer needed. ("example":http://eidiot.github.com/xrope/index.html#FormExampleLink)

h3. How to use it.

* Create a layout group which implements _ILayoutGroup_ and set layout properties.
* Add some display objects or other _ILayoutElement_ to the group.
* Call the _layout()_ method of the group.

h3. Done

* HLineLayout - Horizontal single line layout. ("example":http://eidiot.github.com/xrope/index.html#HLineExampleLink)
* VLineLayout - Vertical single line layout. ("example":http://eidiot.github.com/xrope/index.html#VLineExampleLink)
* HMultiLineLayout - Horizontal multi-line layout. ("example":http://eidiot.github.com/xrope/index.html#HMultiLineExampleLink)
* VMultiLineLayout - Vertical multi-line layout. ("example":http://eidiot.github.com/xrope/index.html#VMultiLineExampleLink)
* HBoxLayout - Horizontal box layout. ("example":http://eidiot.github.com/xrope/index.html#HBoxExampleLink)
* VBoxLayout - Vertical box layout.("example":http://eidiot.github.com/xrope/index.html#VBoxExampleLink)
* AlignLayout - Align position group layout. ("example":http://eidiot.github.com/xrope/index.html#AlignExampleLink)
