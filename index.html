English | 中文
Simple layout library for native as3 display objects.

Features

  • For native as3 display objects. No Flex or any other dependency.
  • AtomLayout align DisplayObject using getBounds(). No worries about register point in FLA.
  • Easily composite layout elements/groups. No more DisplayObjectContainer needed. (example)

How to use

  • Create a layout group which implements ILayoutGroup and set layout properties.
  • Add some display objects or other ILayoutElement to the group.
  • Call the layout() method of the group.

Layout groups (AsDoc)

  • HLineLayout - Horizontal single line layout. (example)
  • VLineLayout - Vertical single line layout. (example)
  • HMultiLineLayout - Horizontal multi-line layout. (example)
  • VMultiLineLayout - Vertical multi-line layout. (example)
  • HBoxLayout - Horizontal box layout. (example)
  • VBoxLayout - Vertical box layout.(example)
  • AlignLayout - Align position group layout. (example)
  • GridLayout - Grid layout. (example)
*Note difference between MultiLineLayout and BoxLayout: Lines in MultiLineLayout has the same height as the lineHeight property, while height of lines in HBoxLayout according to the max height of elements in each line.
